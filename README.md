# MAAIP
基於MAA框架技術的idolypride自動化程序

## 前端框架
neko-para佬的MaaCommonJS

## 後端框架
由Github上最火熱的lk99開源套件[MaaFramework](https://github.com/MaaAssistantArknights/MaaFramework) 製作
